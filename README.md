# ChatECNU Desktop

华东师范大学 AI 超级智能体平台 ([chat.ecnu.edu.cn](https://chat.ecnu.edu.cn)) 的桌面客户端。

## 功能特性

- **独立桌面应用**：无需打开浏览器，一键启动
- **系统托盘**：关闭窗口后最小化到托盘，保持会话不中断
- **自动更新**：启动时自动检查更新，也可通过托盘菜单手动检查

## 下载安装

从 [chat.ecnu.edu.cn](https://chat.ecnu.edu.cn) 主页或 [GitHub Releases](https://github.com/ECNU/ChatECNU-Desktop/releases) 下载最新版本。

- **Windows**：下载 `.exe` 安装程序，双击运行，按提示完成安装
- **macOS**：下载 `.dmg` 文件，打开后将应用拖入 Applications 文件夹
